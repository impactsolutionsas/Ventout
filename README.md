# VENTOUT

E-commerce application for electronics and home appliances, built with React + Vite and Supabase.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` in `.env.local`
3. Run the app:
   `npm run dev`
